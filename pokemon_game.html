Recreating pokemon from scratch using unity!

Built with Unity’s Game engine

  • Implemented inventory system, catching system, battle system, Pokémon party system all while utilizing scriptable objects in Unity
  • Implemented quest system
  • Implemented buy/sell items system
  • Implemented Sound system.
  • Implemented Save & Load system.
  • Used a state machine to handle logic for different game states, made the statemachine a namespace so I can call in their functions across different files. Enter, Execute and Exit functions that will handle logic for entering a state, exiting a state and executing a state. In the State class theres logic for Pushing and Popping.
  • Currently implementing pokemon box functionality
