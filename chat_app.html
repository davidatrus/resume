✨ Key Features

Chatroom application built with React and Socket.io where users can create accounts, add friends, and chat in real time.

  • Real-time messaging between users using WebSockets
  • Create, join, and manage group conversations
  • Persistent user ID saved locally for seamless experience
  • Scalable to unlimited users and messages

📇 Contacts tab: Easily add and manage your chat friends
