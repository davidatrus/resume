Built a Realtime Chat Application Using React and Socket.io

  • Source code

Chatroom application, where you can create your own account, add contacts and instantly message between friends!

So once you first get into webpage you will be on the Login form since you have no info saved locally. Simply press the create ID button and you'll be now moved to the dashboard where you'll see a Convos tab and a Contacts tab, your ID(username) in bottom left corner along with "new convos" or "new contacts" button field depending on what tab you are in.

  • Once created a new contact switch to Convos tab and now you can create a convo with the friend you just added. hit enter and send your message, and there you go you can now chat in real time between you and your friend.
  • No limit on how many users can join, how many people can be in a convo and how many convos you can have. Again to add more people to a convo first you need to add the new person as a friend, click on create room and select which friends you want to add to that room and you can start chatting once again
