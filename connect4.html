✨ Key Features

Connect 4: AI Showdown is a fully-featured two-player game built using React and Node.js. Originally a local-only project, it's grown into a polished, real-time multiplayer experience with dynamic AI and engaging visuals.

You can play against a friend in real time through online multiplayer, choosing to host a private room with a code or a public room that others can join from the lobby. It’s powered by a live Node.js backend using Socket.IO.

Prefer solo play? Challenge a computer opponent with 4 distinct AI difficulty levels — including an Impossible mode powered by the Minimax algorithm with alpha-beta pruning.

The game highlights winning moves, plays sound effects, includes a confetti celebration on victory, and adapts its AI behavior based on your difficulty selection. It also features animations, rematch logic, a polished UI/UX, and a responsive design for play on any screen size.

Whether you’re learning about game theory or just looking for a quick match, give it a shot!

New in v3.0: Introducing Spectator Mode! Friends can now watch live games in real time with synced gameplay, player highlights, and an updated UI that shows who's watching. Perfect for streamers, friends, or just keeping an eye on the competition.
